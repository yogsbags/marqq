## Task 1: Build calibration writer helper

1. Implement `platform/content-engine/calibration-writer.js` with `appendCalibrationNote({agent, companyId, metric, baseline, actual, variance})` and `getLatestCalibrationNote(agent, companyId)`.
2. Notes should live under `platform/crewai/agents/{agent}/memory` and include timestamp + recommended adjustment (e.g., “Raise CTR prediction by 40%”).
3. Add defensive guards so missing directories are created and repeated notes for current variance are deduped (skip if last note has same company+metric+variance).

## Task 2: Wire outcome sink through calibration writer

1. Update `outcome-processor.js` to call `appendCalibrationNote` when `variance_pct > 30`.
2. Update backend `writeOutcomeLedgerRow` helper to attach a note summary to `artifact.data.trigger_context` so `writeTasksCreated` sees it.
3. Document the note format so Phase 08-03 can inject it into prompts.

## Task 3: Add automated coverage

1. Create `platform/content-engine/calibration-writer.test.js` that uses `node:test`, writes to temporary memory files, and asserts notes appear/are deduped.
2. Cover `getLatestCalibrationNote()` reading logic.
3. Run `node --test platform/content-engine/calibration-writer.test.js` as verification.
