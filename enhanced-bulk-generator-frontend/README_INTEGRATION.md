# Enhanced Bulk Generator Integration

This is the Enhanced Bulk Content Generator integrated into the Torqq AI platform.

## Quick Start

1. **Install dependencies:**
   ```bash
   cd enhanced-bulk-generator-frontend
   npm install
   ```

2. **Start the Next.js development server:**
   ```bash
   npm run dev
   ```

   The app will run on **http://localhost:3003**

3. **Access from Torqq AI:**
   - Click on "AI Content" in the sidebar
   - The Enhanced Bulk Generator will be embedded automatically
   - If the app isn't running, you'll see instructions to start it

## Features

- **SEO Research**: Automated SEO data collection
- **Topic Generation**: AI-powered topic suggestions
- **Deep Research**: Comprehensive content research
- **Content Creation**: Bulk content generation
- **Content Validation**: Quality checks and validation
- **SEO Optimization**: Automatic SEO improvements
- **Publication**: Multi-platform publishing
- **Backend Integration**: Full backend support included

- Workflow orchestrator
- Content creator and validator
- SEO optimizer
- Research tools
- Google APIs integration

Make sure to set up the required environment variables. See the original README.md for details.

## Integration with Torqq AI

The Enhanced Bulk Generator is embedded in an iframe when you click "AI Content" in the Torqq AI menu. If the Next.js app isn't running, you can:

1. Start it using the instructions above
2. Click "Refresh" in the Torqq AI interface
3. Or click "Open in New Tab" to use it standalone
